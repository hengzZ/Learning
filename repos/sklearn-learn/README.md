# sklearn-learn #

## scikit-learn install ##
Scikit-learn requires:

    * Python (>= 2.6 or >= 3.3),
    * NumPy (>= 1.6.1),
    * SciPy (>= 0.9).

```
pip install -U scikit-learn
```
or  
```
sudo apt-get install python-sklearn
```
